# ShopVibe — Design System

A design system crafted for fashion and lifestyle e-commerce storefronts that demand visual punch and effortless conversion. The spacious layout philosophy ensures products have the breathing room to shine, with generous whitespace framing hero imagery. A bold fuchsia primary drives urgency on CTAs, while cyan and yellow accents inject youthful energy.

## Color Palette

- **Primary** (#D946EF): The main brand color used for primary buttons, links, and key interactive elements throughout the storefront.
- **Secondary** (#22D3EE): Accent color for secondary actions, sale tags, and promotional pricing highlights.
- **Tertiary** (#FACC15): Highlight color for badges, notification dots, and attention-grabbing callouts.
- **Background** (#FAFAFA): The default page and surface background providing a clean canvas.
- **Surface** (#FFFFFF): Card and elevated surface background, slightly brighter than the page.
- **Text Primary** (#17191C): Main body text and headings — high contrast on light backgrounds.
- **Text Secondary** (#6B7280): Supporting body text, metadata, and subtle descriptions.
- **Text Tertiary** (#9CA3AF): Placeholder text, disabled states, and fine print.
- **Border** (#E5E7EB): Default border color for cards, inputs, and dividers.

## Typography

- **Display**: Poppins 56px extra-bold, 1.1 line height, 0.02em tracking. Hero banners and splash headlines.
- **Headline**: Poppins 40px bold, 1.2 line height, 0.01em tracking. Section headers and product titles.
- **Subhead**: Poppins 26px semi-bold, 1.3 line height. Subsection titles and featured descriptions.
- **Body Large**: Nunito 18px regular, 1.6 line height. Intro paragraphs and long-form product descriptions.
- **Body**: Nunito 16px regular, 1.5 line height. Standard body text throughout the interface.
- **Body Small**: Nunito 14px regular, 1.4 line height. Secondary information and compact layouts.
- **Caption**: Nunito 12px medium, 1.3 line height. Labels, tags, and small annotations.
- **Overline**: Nunito 11px bold, 1.2 line height, 0.08em tracking. Category labels above product names.
- **Code**: Space Mono 14px regular, 1.5 line height. Inline code snippets and monospace data.

## Corner Radius

- **Small**: 4px — inputs, chips, small badges
- **Medium**: 12px — cards, modals, dropdown menus
- **Large**: 16px — prominent cards, featured sections
- **Extra Large**: 24px — hero cards and full-width containers
- **Pill**: 9999px — buttons, tags, and pill-shaped elements
- **None**: 0px — sharp edges for dividers and full-bleed images

## Spacing

- **xs**: 4px
- **sm**: 8px
- **md**: 12px
- **lg**: 16px
- **xl**: 24px

- **3xl**: 48px
- **4xl**: 64px
- **5xl**: 96px

## Elevation & Depth

- **Subtle**: 0 1px 2px rgba(0,0,0,0.05) — resting cards and default surfaces
- **Medium**: 0 4px 6px rgba(0,0,0,0.07), 0 2px 4px rgba(0,0,0,0.06) — hovered cards and dropdowns
- **Large**: 0 10px 15px rgba(0,0,0,0.1), 0 4px 6px rgba(0,0,0,0.05) — modals and overlays
- **Overlay**: 0 20px 25px rgba(0,0,0,0.15), 0 8px 10px rgba(0,0,0,0.08) — drawer panels and popovers

## Components

### Buttons

- **Primary**: Fuchsia background (#D946EF), white text, pill shape, 12px 24px padding. Main action buttons.
- **Secondary**: Transparent background, #D946EF text, 1px #D946EF border, pill shape. Alternative actions.
- **Ghost**: Transparent background, Text Secondary (#6B7280), no border. Tertiary actions.

### Cards

- **Product Card**: White background (#FFFFFF), 12px border radius, Medium elevation. Product grids and listings.
- **Feature Card**: White background, 16px border radius, Large elevation, 24px padding. Featured items and promotions.

### Inputs

- **Text Input**: White background, 1px Border (#E5E7EB), 4px radius, 12px 16px padding, Text Primary label.
- **Search Input**: White background with subtle shadow, 9999px radius, magnifying glass icon. Header search bar.
